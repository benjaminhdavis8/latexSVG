% Prettyref relies on the use of a convention in the creation and use of
% labels. For example, a convention where sections are labeled with
% sec:<unique-id> would allow us to use prettyref to build all referances to a
% section in a consistent manner.
% The effect of the below is to set up the \prettyref command to infer the
% presentation of a label <convention>:<unique-id> by looking at the
% <convention> part.
